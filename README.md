 
СберКалендарь
Интеллектуальный помощник для планирования IT-мероприятий в Северо-Западном регионе

================================================================

О ПРОЕКТЕ

СберКалендарь — это корпоративный AI-ассистент для поиска, фильтрации и планирования IT-мероприятий (хакатонов, митапов, конференций) в регионах Северо-Запада России.

Возможности:

Семантический поиск по смыслу (не по ключевым словам)
Автоматическое распознавание городов: СПб, Калининград, Мурманск и др.
Интерактивный календарь с визуализацией событий
Ролевая модель: администратор → руководитель → пользователь
Автоматическая отправка email при регистрации с логином, паролем и ролью
Защита от не-IT запросов через GigaChat
================================================================

ТЕХНОЛОГИИ

Backend: Python, Flask
AI: sentence-transformers (LaBSE), GigaChat API
Хранение: Excel (log.xlsx), JSON (events_vector_db.json)
Frontend: HTML, CSS, JavaScript (дизайн в стиле GitVerse / Сбера)
================================================================

УСТАНОВКА

Клонируйте репозиторий:
git clone https://github.com/ваш-логин/sber-calendar.git
Создайте виртуальное окружение:
python -m venv .venv
.venv\Scripts\activate (Windows)
Установите зависимости:
pip install -r requirements.txt
Подготовьте данные:
log.xlsx — с колонками: login, pasword, type, fio, email, manager
events_vector_db.json — векторная база событий
Настройте GigaChat (опционально):
Получите GIGACHAT_CREDENTIALS
Вставьте в app.py
Запустите:
python admin/app.py # админка (порт 5001)
python app.py # чат (порт 5000)
Откройте в браузере:
Админка: http://localhost:5001
Чат: http://localhost:5000
================================================================

БЕЗОПАСНОСТЬ

Вход в админку только для пользователей с type = "admin"
Пароли временные (хранятся в Excel — для продакшена рекомендуется хеширование)
Email отправляется через Gmail с паролем приложения (безопасно)
Все данные хранятся локально
